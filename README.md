# Dotfiles
My personal dotfiles

## Dependencies
* Vim
* stow
* git
* curl
* python3
* cmake
* clangd

## Installation
```
mkdir be4zad-dotfiles
git -C be4zad-dotfiles clone https://github.com/be4zad/dotfiles
cd be4zad-dotfiles
```

Then:
```
make install
```

For uninstall:
```
make uninstall
```
